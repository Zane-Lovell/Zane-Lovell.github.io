During my financial Econometrics course at Georgetown University Dr. Jose Luis Guerrero Cusumano showed our class a forecasting model for the spread of COVID19 in Italy. I was able to apply this model to the spread of COVID19 cased in New York State. The image in the top left position shows the data in it's raw CSV form. I was able to filter through the data with a simple excel pivot table in order to find data for only New York State. Next I was able to start apply Dr. Guerro's model which fits a quadratic equation to the natural log of the cummulative number of cases.
